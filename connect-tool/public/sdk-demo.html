Camaleonic Connect — SDK demo

Mint an SDK token server-side, paste it here, then click one of the platform buttons. The widget opens a popup, the end-user authorizes, and we log the resulting onSuccess payload below.

Instagram is reached through the Facebook flow — connect a Facebook Page that owns the IG Business Account and tick "Include Instagram" on the page picker.

Drop into your own site

<script src="https://smconnector.camaleonicanalytics.com/connect-sdk.js"></script>
<script>
  // sdkToken is minted SERVER-SIDE with your cmlk_live_* API key:
  //   POST /v1/sdk-tokens  { user_id: "your-end-user-id", ttl: 1800 }
  const handle = CamaleonicConnect.init({
    sdkToken: SDK_TOKEN_FROM_YOUR_BACKEND,
    workspace: "your-slug",
    platforms: ["twitch", "instagram"],
    onSuccess: ({ accountIds, platform }) => { /* persist accountIds */ },
    onError:   (err) => console.warn(err),
    onExit:    ()    => console.log("user closed"),
  });
  document.getElementById("connect").onclick = () => handle.open("twitch");
</script>

Prefer npm? npm i @camaleonic/connect then import CamaleonicConnect from "@camaleonic/connect". TypeScript types ship alongside the script tag at /connect-sdk.d.ts.

Server reference: openapi.yaml · interactive docs at /docs.html.
